## In this release:

### <a href="https://vvwojxr.clicks.mlsend.com/tj/cl/eyJ2Ijoie1wiYVwiOjE1MDAxNTAsXCJsXCI6MTgwMDMwODYzMzU4MDM1OTU3LFwiclwiOjE4MDAzMDg3NDE4NjY3OTM4MX0iLCJzIjoiZmIxNzgzYThhYTRkODE3NCJ9" target="_blank">New Landing Page</a>

You'll see the landing page for the app is completely overhauled. We've designed it to allow the pilot to easily verify that all systems (Drone Battery/Signal, Drone sensors, Controller Battery/Signal, CalTopo, Wifi, etc) are operational before flying.

### <a href="https://vvwojxr.clicks.mlsend.com/tj/cl/eyJ2Ijoie1wiYVwiOjE1MDAxNTAsXCJsXCI6MTgwMDMwODYzMzY1Mzc1OTkzLFwiclwiOjE4MDAzMDg3NDE4NjY3OTM4MX0iLCJzIjoiYzkyNWJmZDk5N2E4ODczNyJ9" target="_blank">AirSense</a>

For drones with AirSense, we visually warn the pilot of nearby aircraft broadcasting on ADS-B. Note: not all aircraft broadcast ADS-B, so the pilot must still maintain visual airspace awareness.

### <a href="https://vvwojxr.clicks.mlsend.com/tj/cl/eyJ2Ijoie1wiYVwiOjE1MDAxNTAsXCJsXCI6MTgwMDMwODYzMzcyNzE1MDIyLFwiclwiOjE4MDAzMDg3NDE4NjY3OTM4MX0iLCJzIjoiNjNmNGQ5YTg5NTgxMThiNiJ9" target="_blank">New Map Interface</a>

Much better interface for viewing and interacting with map objects.

### <a href="https://vvwojxr.clicks.mlsend.com/tj/cl/eyJ2Ijoie1wiYVwiOjE1MDAxNTAsXCJsXCI6MTgwMDMwODYzMzgyMTUyMjA4LFwiclwiOjE4MDAzMDg3NDE4NjY3OTM4MX0iLCJzIjoiOGM2MDE5MmM5NTZjOTBjZCJ9" target="_blank">DJI Health Monitoring/Warnings</a>

We bring all DJI warnings into the app, and bring in a home-screen "health check" for the pilot to verify all subsystems of their drone.

### <a href="https://vvwojxr.clicks.mlsend.com/tj/cl/eyJ2Ijoie1wiYVwiOjE1MDAxNTAsXCJsXCI6MTgwMDMwODYzMzg4NDQzNjY3LFwiclwiOjE4MDAzMDg3NDE4NjY3OTM4MX0iLCJzIjoiMWE5M2RjMjUxMTJlZDI4OSJ9" target="_blank">Procedures Checklist</a>

You can now set up your controller with SOPs and checklists for easy pilot reference.

### <a href="https://vvwojxr.clicks.mlsend.com/tj/cl/eyJ2Ijoie1wiYVwiOjE1MDAxNTAsXCJsXCI6MTgwMDMwODYzMzk2ODMyMjc5LFwiclwiOjE4MDAzMDg3NDE4NjY3OTM4MX0iLCJzIjoiNjRmMzcxOWZlYTZmZTQ5MCJ9" target="_blank">Background Mode</a>

If you're flying a drone without DJI SDK support (like the Mini 5 for example) or you prefer to fly in the native DJI app, you can turn on "Background mode" - which lets you track to CalTopo and livestream while flying with the native DJI Piloting app instead of Eagle Eyes Pilot. You can find this in "..." ‚Üí Set Up Controller ‚Üí Advanced Options.

### Custom Buttons

From the piloting view you can go to the menu, settings, button customization - allowing you to configure custom button actions to suit your workflow.

## Other improvements/fixes since 1.5.0

- **Joining non-team CalTopo maps by QR Code.** Previously if you drone-scanned a CalTopo QR code not belonging to your team, you were directed to log in manually. That is now fixed, and you should be able to join the map right away.

- **Share your realtime drone location over the internet** to other Eagle Eyes Pilot users connected to your CalTopo map. You can turn on "Use CalTopo Map for MQTT" in settings. This is an experimental feature and we may change the way it is implemented in the future, so we have kept it hidden for now. It will be officially released in 1.7. In the meantime, you are welcome to try it and we would like to hear if it is useful to you.
