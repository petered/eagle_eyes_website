How it works

  1. Select or drag your DJI Matrice 4 series .MP4 video files
  2. The tool extracts embedded subtitle/telemetry data from each file
  3. Converts the telemetry to the classic DJI SRT format
  4. Downloads the generated .SRT files to your computer

All processing happens locally in your browser — no files are uploaded to any server.
