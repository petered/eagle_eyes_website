• 1
    Locate the downloaded APK file Once your download completes on your device, locate the downloaded .APK file which should be in your Downloads folder in the files of your device.
  • 2
    Install the APK Open that .APK file and install the APK. If it's blocked, enable "Install from Unknown Sources" in your settings.
  • 3
    Open Eagle Eyes Pilot app and allow permissions Open Eagle Eyes Pilot app once it's finished installing and allow permissions. When asked for location access, set it to "While using the app".
  • 4
    Optional device configuration You can configure the app by selecting Setup on Controller in the settings accessible from the three dots (...). Choose a default CalTopo map, a default drone name, and download any offline maps. Then you're ready to either start flying or using the AirOps map in Eagle Eyes Pilot!
