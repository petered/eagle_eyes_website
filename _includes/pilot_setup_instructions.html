Installing the Eagle Eyes app onto your drone controller or Android device
Click to watch
  • 1
    Locate the downloaded APK file Once your download completes on your device, locate the downloaded .APK file which should be in your Downloads folder in the files of your device.
  • 2
    Install the APK Open that .APK file and install the APK. If it's blocked, enable "Install from Unknown Sources" in your settings.
  • 3
    Open Eagle Eyes Pilot app and allow permissions. Open Eagle Eyes Pilot app once it's finished installing and allow permissions. When asked for location access, set it to "While using the app".
  • 4
    Activate your Eagle Eyes License. See the License Activation walk through in the video linked above for a walk through of the License Activation. Basic Eagle Eyes features can be accessed for free without license activation.
  • 5
    Optional device configuration You can configure the app by selecting Setup on Controller in the settings accessible from the three dots (...). Activate License (if you haven't already), connect a CalTopo Service Account, choose a default CalTopo map, choose a default drone name, and download any offline maps. See the video linked at the top of these instructions for a walk through of this set up. Then you're ready to either start flying or using the AirOps map in Eagle Eyes Pilot!
