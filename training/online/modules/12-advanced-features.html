Advanced Features

8.1 ADS-B Aircraft Tracking

Eagle Eyes integrates with DJI AirSense to provide real-time awareness of manned aircraft in your vicinity.

⚠️
AirSense Requirements: ADS-B tracking requires a DJI drone with AirSense capability (Mavic 3 Enterprise, M30 series). Not all drones support this feature.

Accessing Aircraft View

  1. Open the hamburger menu (☰) on the main screen
  2. Select View ADS-B Aircraft
  3. The aircraft map view opens with all detected aircraft
📷

Screenshot: ADS-B Aircraft Map View showing detected aircraft

Aircraft Information

Tap any aircraft icon on the map to view details:

FieldDescription
Call SignAircraft identifier (ICAO code or registration)
AltitudeHeight above ground level
DistanceDistance from your drone
HeadingDirection of travel
SpeedGround speed (when available)

Alert Levels

LevelDistanceResponse
ℹ️ Info>2 kmAwareness only, continue operations
⚠️ Caution1-2 kmIncrease vigilance, prepare to yield
🔶 Warning500m-1 kmConsider descending or landing
🛑 Critical<500mLand immediately if safe

OpenSky Lookup

For aircraft with valid ICAO codes, Eagle Eyes can look up additional information from OpenSky Network:

  • Aircraft registration and type
  • Operator information
  • Flight history

Managing Aircraft Alerts

For each detected aircraft, you can:

  • Mute - Silence alerts for this specific aircraft
  • Hide - Remove from map display temporarily
  • Info - View full aircraft details
💡
If coordinating with a known helicopter during SAR, mute its alerts to reduce noise while maintaining awareness of other aircraft.

8.3 Device Health Monitoring

The Device Health system monitors your drone and controller for potential issues during operations.

Accessing Device Health

  1. Tap the Device Health card on the home page
  2. Or tap the warning banner if issues are detected

Health Categories

CategoryWhat It Monitors
🛰️ GPSSatellite count, signal quality, accuracy
💾 StorageSD card space, controller storage
🔋 BatteryDrone and controller battery levels
⚙️ RotorsMotor status, wear indicators
📡 SignalRC link quality, video transmission
🌡️ TemperatureBattery and motor temperatures

Storage Warnings

SD card space warnings use tiered thresholds:

LevelRemaining SpaceAction
✅ OK>2 GBNormal operation
⚠️ Low500MB - 2GBConsider clearing old files
🛑 Critical<500 MBClear space before flight

GPS Warnings

GPS quality affects navigation accuracy:

  • Weak Signal - Move to open area, wait for more satellites
  • No GPS - Cannot fly autonomous missions or return-to-home
  • Compass Error - Calibration needed (see Module 1)

Resolving "Check Notices"

When multiple issues exist, the card shows "Check Notices". Tap to see all warnings:

  1. Review each warning category
  2. Address highest priority issues first
  3. Some warnings may require postponing flight
⚠️
Never ignore red/critical warnings before flight. They indicate conditions that could result in loss of aircraft or mission failure.

8.4 Team Procedures

The Procedures system provides quick access to your team's SAR standard operating procedures.

Accessing Procedures

  1. Tap the Procedures card on the home page
  2. Or open Settings → Procedures

Viewing Procedures

The procedures dialog displays:

  • Custom procedure text (configured by team lead)
  • Quick reference checklists
  • Links to external documents

Configuring Procedures

  1. Open Settings → Procedures
  2. Tap Edit Procedure Text
  3. Enter your team's procedures
  4. Add external links if needed
  5. Tap Save

Suggested Procedure Content

  • Pre-flight safety checklist
  • Detection confirmation protocol
  • Communication procedures
  • Emergency contacts
  • Handoff procedures
💡
Configure procedures with links to your team's Google Drive or SharePoint documents for easy access to full SOPs during operations.

8.5 DJI Log Sync

Eagle Eyes can automatically sync your DJI flight logs for analysis with services like Airdata.

How It Works

  1. Eagle Eyes monitors DJI log directories
  2. New logs are detected automatically
  3. Logs are organized for easy export
  4. Upload to Airdata or other services

Log Upload Dialog

  1. Open Settings → Flight Logs
  2. View available logs by date
  3. Select logs to upload
  4. Tap Upload or Export
📷

Screenshot: DJI Log Upload Dialog

Monitored Directories

Eagle Eyes monitors these standard DJI log locations:

  • /DJI/LOG/ - Main flight logs
  • /DJI/FlightRecord/ - Detailed records
  • Controller internal storage logs

Airdata Integration

For automatic Airdata sync:

  1. Install Airdata UAV app
  2. Configure sync folder in Airdata
  3. Eagle Eyes copies logs to sync folder
  4. Airdata uploads automatically
ℹ️
Flight logs are essential for post-mission analysis, regulatory compliance, and insurance purposes. Always ensure logs are backed up.

8.6 Background Mode

Background Mode allows Eagle Eyes to operate alongside DJI Pilot or DJI Fly apps.

When to Use Background Mode

  • Using DJI Pilot 2 for waypoint missions
  • Need features only available in DJI apps
  • Running detection while another app controls drone
  • Livestreaming via Eagle Eyes while flying with DJI app

Enabling Background Mode

  1. Open Settings → General
  2. Toggle Background Mode to ON
  3. Confirm the first warning dialog
  4. Confirm the second warning dialog
  5. Restart Eagle Eyes when prompted
⚠️
Double Confirmation Required: Background mode requires confirming two separate dialogs to prevent accidental activation.

Operational Workflow

  1. Start Eagle Eyes in background mode
  2. Connect drone via USB
  3. Switch to DJI Pilot/Fly app
  4. Fly using DJI app controls
  5. Eagle Eyes continues detection and telemetry
  6. Switch back to Eagle Eyes to view detections

Limitations in Background Mode

FeatureStatus
AI Detection✅ Full functionality
Telemetry Broadcast✅ Full functionality
CalTopo Sync✅ Full functionality
Livestreaming✅ Works via screen capture
Camera Control❌ Controlled by DJI app
Gimbal Control❌ Controlled by DJI app
Flight Missions❌ Use DJI app

Screen Capture for Livestreaming

In background mode, livestreaming uses screen capture:

  1. Grant screen capture permission when prompted
  2. Switch to DJI app with video feed visible
  3. Eagle Eyes captures and streams the screen
  4. Viewers see the DJI app interface

8.7 Update Checker

Eagle Eyes includes automatic update checking to ensure you always have the latest features and fixes.

Automatic Updates

Eagle Eyes checks for updates:

  • On app startup
  • Every 24 hours while running
  • When you tap the version card

Version Card Status

StatusMeaning
🟢 CurrentYou have the latest version
🔵 Update AvailableNew version ready to download
⚠️ Update RequiredCritical update needed for continued operation

Update Channels

ChannelDescription
StableTested, production-ready releases (recommended)
BetaPreview features, may contain bugs

Switching Channels

  1. Open Settings → About
  2. Tap Update Channel
  3. Select Stable or Beta
  4. Restart app to apply

Manual Download

  1. Open Settings → About
  2. Tap Check for Updates
  3. If available, tap Download
  4. Open downloaded APK to install

Team Update Coordination

💡
Best Practice: Coordinate updates with your team:
  • All team members should use the same version
  • Test updates before critical missions
  • Never update during an active operation
