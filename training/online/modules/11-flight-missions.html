Module 7

Flight Missions

⏱️ 75 minutes

Learning Objectives

  • Create flight grid missions for systematic area searches
  • Choose the right search pattern for each scenario
  • Create line-based missions (Follow Line and Corridor Search)
  • Configure terrain following for consistent altitude above ground
  • Save, load, and resume missions
  • Execute and monitor autonomous missions
  • Handle mission interruptions, RTH events, and battery changes

7.1 Introduction to Flight Missions

Eagle Eyes provides powerful autonomous flight capabilities for systematic SAR operations:

  • Flight Grid Planning — Create search patterns over polygons (Grid, Expanding Square, Star)
  • Line Missions — Follow Line and Corridor Search along drawn polylines
  • Fly-to-Waypoint — Quick flights to specific locations
  • Mission Library — Save, load, and resume missions across flights
  • Mission Control — Execute and monitor missions in-app (SDK5)
  • KMZ Export — Export for DJI Pilot 2 (SDK4 fallback)

All mission settings are automatically saved between sessions, so you don't need to re-enter parameters for similar missions.

7.2 Flight Grid Planning

Create systematic coverage patterns over search areas.

Accessing the Dialog

  2. Tap the polygon to select it
  3. Tap menu (⋮) or right-click

7.3 Search Patterns

Eagle Eyes supports multiple search patterns optimized for different scenarios.

Grid Pattern (Parallel Lines)

Best For: Systematic area searches, mapping, maximum coverage

  • Parallel flight lines covering the entire polygon area (boustrophedon/lawn-mower pattern)
  • Even coverage throughout
  • Predictable path, easy to track progress
  • Configurable orientation and start corner

Expanding Square Pattern

Best For: Point Last Seen (PLS) searches, unknown direction of travel

  • Spiral from center outward
  • Covers high-probability areas first
  • Expands systematically in all directions

Star Pattern (Radial Spokes)

Best For: Trail intersections, 360° quick coverage

  • Radial spokes from center point
  • Choose between 6 spokes or 10 spokes via dropdown
  • Quick coverage in all directions
  • Best combined with other patterns for thorough coverage

Pattern Selection Guide

ScenarioPatternReason
Lost hiker, unknown directionExpanding SquareStarts at PLS, covers all directions
Known search areaGridSystematic, complete coverage
Trail junctionStarQuick coverage of all trails
Water search (known current)Grid (oriented)Covers drift zone
Road, river, or trailFollow Line / CorridorFollows the feature directly (see 7.4)

7.4 Line Missions: Follow Line & Corridor Search

For linear features like roads, rivers, trails, or ridgelines, line-based missions are more efficient than grid patterns.

Follow Line

Fly along any polyline drawn on the map. The drone follows the line point-by-point at your specified altitude and speed.

  1. Draw or select a polyline on the map
  2. Tap the line and select Create Line Mission
  3. Choose Follow Line mode
  4. Set altitude, search speed, transit speed, gimbal pitch, and camera mode
  5. Optionally check Reverse Direction to fly the line backwards

Corridor Search

Like Follow Line, but with a configurable corridor width. The drone flies a search pattern along the line, sweeping a defined width on either side. This is ideal for searching along a trail where a subject may have wandered off-path.

  1. Draw or select a polyline on the map
  2. Choose Corridor Search mode
  3. Set the corridor width (how far to each side of the line to search)
  4. Set line spacing within the corridor
  5. Configure remaining parameters as with any grid mission
💡
Use Follow Line for quick passes along a feature. Use Corridor Search when you need to sweep a wider area centered on a linear feature like a trail or riverbank.

7.5 Terrain Following

In hilly or mountainous terrain, a fixed altitude above takeoff can mean the drone is too high over valleys and too low over ridges. Terrain following solves this by adjusting each waypoint's altitude based on the ground elevation beneath it.

How It Works

  • Toggle AGL (Above Ground Level) mode in the mission dialog
  • When you generate or fly the mission, Eagle Eyes queries an elevation service to get the ground height at each waypoint
  • Each waypoint altitude is adjusted so the drone maintains a consistent height above the terrain
  • If any waypoint would exceed the drone's maximum altitude limit, a warning is displayed

When to Use

TerrainModeWhy
Flat areaRelative altitudeNo significant terrain variation — simpler
Hills, valleys, slopesAGLMaintains consistent search altitude above ground for even coverage
Mountains with large elevation changesAGLPrevents the drone from being too high/low over terrain features
⚠️
Terrain following requires an internet connection to query elevation data at mission creation time. If elevation data is unavailable, the mission exports without terrain adjustment. Always verify the planned altitudes look reasonable on the map preview.

7.6 Fly-to-Waypoint Missions

Simple point-to-point flights for investigating detections or reaching specific locations quickly.

Creating a Fly-to-Waypoint Mission

  1. Long-press on the map at your destination
  2. Select Fly to Waypoint
  3. Set altitude and speed
  4. Choose a waypoint action
  5. Tap Fly

Waypoint Actions

ActionDescription

💡

7.7 Mission Library

Eagle Eyes automatically saves missions when you generate or fly them. The Mission Library lets you manage, reload, and resume saved missions.

What Gets Saved

  • Mission name (auto-filled from polygon/line name, editable)
  • Pattern type (Grid, Expanding Square, Star, Follow Line, Corridor)
  • All flight parameters (altitude, speed, overlap, gimbal, etc.)
  • The search area polygon points
  • KMZ file for the mission
  • Progress state (which waypoint you paused at, if applicable)

Accessing the Mission Library

Open from the Fly Screen menu. Missions are organized by session — you see only missions from the current session.

Mission Status Indicators

BadgeMeaning
CompletedMission has been fully flown
Paused at 42/100Mission was paused or interrupted — shows the resume point
No badgeFresh mission, not yet flown

Actions

  • Tap a mission to load it. If it has a pause point, you can resume from where you left off.
  • Pencil icon to rename the mission
  • Trash icon to delete
💡
If duplicate names are detected in the same session, Eagle Eyes automatically appends "(2)", "(3)", etc. Give missions descriptive names like "North Ridge Grid 50m" to make them easy to find later.

7.8 Mission Execution (SDK5)

For SDK5 drones (Mavic 3 Enterprise, M30 series, M4 series), execute missions directly from Eagle Eyes.

Pre-Flight Checklist

ItemRequirement
Drone ConnectedActive connection to drone
GPS Lock4+ satellites acquired
Battery LevelSufficient for mission
Home Point SetValid home location

All items must be green before starting.

Control Buttons

ButtonFunction
UploadTransfer mission to drone
StartBegin mission execution
PausePause at current waypoint (button turns blue)
ResumeContinue from the exact waypoint where you paused
StopAbort mission and return to home
RestartStart mission from the beginning

Mission Progress Overlay

During flight, a real-time overlay shows:

  • Waypoint counter: "WP 42/100"
  • Progress percentage: "42%"
  • Color-coded status indicator
  • Pause, Resume, and Stop buttons

Execution Steps

  1. Open Mission Control after creating or loading a mission
  2. Verify all checklist items are green
  3. Tap Upload and wait for completion
  4. Tap Start to begin
  5. Monitor progress on the map and overlay
  6. Use Pause/Resume as needed
  7. Mission completes automatically and executes your chosen Finish Action
⚠️
Always monitor the mission and maintain visual line of sight. Be ready to pause or stop at any time.

7.9 KMZ Export & DJI Pilot 2

For SDK4 drones or as a backup, export missions as KMZ files compatible with DJI Pilot 2.

Exporting a Mission

  1. Create a flight grid, line mission, or waypoint mission
  2. Tap Export to KMZ
  3. File saves to the Downloads folder with a timestamped filename

Using with DJI Pilot 2

  1. Open DJI Pilot 2
  2. Go to Flight → Mission
  3. Tap Import
  4. Select the exported KMZ file
  5. Review and execute in DJI Pilot 2
💡
KMZ files can also be viewed in Google Earth for mission visualization and briefing purposes.

7.10 Handling Interruptions

Pause & Resume

  • Tap Pause to stop at the current waypoint — the button turns blue to confirm
  • The drone hovers and maintains altitude
  • You can take manual control while paused
  • Tap Resume to continue from the exact waypoint where you paused
  • The pause point (breakpoint) is saved to the database and persists for 24 hours

Resume After Battery Change

If you need to land for a battery swap mid-mission:

  1. Tap Pause to save your progress
  2. Return to home, swap the battery, and take off again
  3. Open the Mission Library and tap the paused mission
  4. The mission resumes from the waypoint where you left off — it regenerates the remaining KMZ from the resume point
💡
The breakpoint persists for 24 hours, so you can resume even after closing and reopening the app.

RTH During Mission

If Return-to-Home triggers during a mission (low battery, signal loss, or manual RTH):

  • The mission pauses automatically and the state shows "Paused by RTH"
  • The breakpoint is saved so you know exactly where to resume
  • After the drone lands and you're ready, load the mission from the library and Resume or Restart
⚠️
Eagle Eyes correctly distinguishes between "mission completed" and "mission interrupted by RTH." If the drone returns home mid-mission, the status will show Paused (not Completed), so you always know there is remaining work.

Loss of Signal

If the radio link is lost during a mission, the drone follows the RC Lost Action you configured:

  • Return to Home — drone flies back to the home point
  • Hover — drone holds position and waits for signal recovery
  • Land — drone lands at current position
