Module 1

Getting Started

⏱️ 30 minutes

Learning Objectives

After completing this module, you will be able to:

  • Install Eagle Eyes Pilot on your Android device
  • Grant the required permissions
  • Connect Eagle Eyes to your DJI drone
  • Complete initial configuration for SAR operations

1.1 Download, Setup & Configuration

System Requirements

Requirement Minimum Recommended
Android Version 8.0 (Oreo) 11.0 or newer
RAM 4 GB 6+ GB
Storage 500 MB free 2+ GB free
Screen 5" display 7"+ tablet
💡
Larger tablets provide a better experience for viewing the video feed and map simultaneously.

Installation Steps

Download the App

Eagle Eyes Pilot is distributed directly by your SAR organization. You'll receive either an APK file or a link to a private app store.

Enable Unknown Sources

If installing from an APK: Go to Settings → Security and enable Install unknown apps for your file manager.

Install the Application

Open the APK file and tap Install. Wait for installation to complete, then tap Open.

Grant Permissions

When prompted, grant these permissions:

  • Location - For geotagging detections
  • Camera - For QR code scanning
  • Storage - For saving detections and maps
  • Nearby Devices - For drone communication
⚠️
Only install Eagle Eyes from trusted sources provided by your organization.

Essential Settings

Device Name

Set a unique name so teammates can identify your drone (e.g., "SAR-Drone-1").

Coordinate System

Format Example Common Use
DD 38.889500, -77.035278 General mapping
DDM 38° 53.370' N, 77° 2.117' W Marine/aviation
UTM 18S 323394 4307396 Military/SAR grids
MGRS 18SUJ2339407396 Military operations
⚠️
Coordinate format mismatches between team members can cause search area errors. Confirm your team's standard format.

Offline Maps

Load MBTiles files before heading to the field for areas with poor cell coverage:

  1. Go to ☰ Menu → Map Loader
  2. Tap Select MBTiles
  3. Choose your pre-downloaded map file

Configuration Checklist

  • ☐ Device name is set and identifiable
  • ☐ Coordinate system matches team standard
  • ☐ Units (metric/imperial) match team standard
  • ☐ DJI log folder is configured
  • ☐ Offline maps for search area are loaded
  • ☐ License is active (for AI detection)
