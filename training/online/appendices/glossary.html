Reference

Glossary of Terms

A

AHL (Above Home Level) - Altitude measured from the takeoff point.

AirSense - DJI technology that detects nearby manned aircraft.

ASL (Above Sea Level) - Altitude measured from mean sea level.

ATTI Mode - Flight mode when GPS is unavailable; drone drifts with wind.

B-C

Background Mode - Eagle Eyes runs alongside DJI Pilot, reading telemetry from logs.

Bounding Box - Rectangle drawn around a detected object.

CalTopo - Online mapping platform for SAR teams.

Confidence Score - Percentage indicating AI certainty about a detection.

D-M

DD - Decimal Degrees coordinate format (38.889500, -77.035278)

DDM - Degrees Decimal Minutes format

DMS - Degrees Minutes Seconds format

MBTiles - File format for offline map tiles.

MGRS - Military Grid Reference System coordinate format.

R-U

RTH - Return to Home; automatic flight back to takeoff point.

SAR - Search and Rescue operations.

UDP - User Datagram Protocol; used for drone telemetry sharing (port 7742).

UTM - Universal Transverse Mercator coordinate format.

W

WebRTC - Technology for real-time video streaming.

Wide Camera - Standard RGB camera with wide field of view.
