Troubleshooting Guide

Connection Issues

ProblemSolution
Drone not foundCheck controller power; try different USB cable
SDK registration failedEnsure internet connection; DJI requires online activation
Video feed blackRemove lens cap; try switching cameras

Detection Issues

ProblemSolution
No detections appearingVerify mode is not OFF; check threshold isn't too high
Too many false positivesSwitch to NORMAL mode; increase threshold
Detection lagUse smaller model; close background apps

Map Issues

ProblemSolution
Map not loadingCheck internet or load offline MBTiles
Offline maps not workingVerify MBTiles file isn't corrupted
Wrong positionWait for GPS lock (10+ satellites)

CalTopo Issues

ProblemSolution
Can't connectImprove lighting for QR; try manual entry
Features not syncingCheck connection status; force sync

Streaming Issues

ProblemSolution
Stream won't startCheck internet connection and firewall
Poor qualityLower resolution; improve network
